Bienvenido al sistema

Este sitio web permite a los ciudadanos reportar fallas en el alumbrado público de su colonia o comunidad. El objetivo principal es facilitar la comunicación entre la ciudadanía y el área encargada del mantenimiento urbano.

Hacer un reporte

Objetivo del proyecto

El objetivo de este proyecto es diseñar una plataforma web sencilla y funcional que permita registrar, consultar y dar seguimiento a reportes relacionados con fallas en luminarias públicas.

Mediante este sistema, los ciudadanos pueden informar sobre calles con poca iluminación, lámparas dañadas, postes en mal estado o fallas eléctricas visibles.

Problema que atiende

En muchas colonias existen luminarias apagadas o dañadas que pueden afectar la seguridad de las personas, dificultar el tránsito nocturno y generar una mala imagen urbana. Este sitio busca apoyar el registro organizado de dichas fallas.

Reportes rápidos

El ciudadano puede registrar una falla indicando su ubicación, tipo de problema y una breve descripción.

Consulta de reportes

El sistema permite consultar ejemplos de reportes y revisar su estado: pendiente, en proceso o atendido.

Mejor organización

La información registrada ayuda a identificar zonas con mayor número de fallas y facilita la toma de decisiones.
