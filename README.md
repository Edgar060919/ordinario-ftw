# Sistema de Reporte de Alumbrado Público

## Descripción del proyecto

Este proyecto consiste en un sitio web para registrar, consultar y administrar reportes de fallas en el alumbrado público.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- XML
- GitHub Pages

## Páginas del sitio

1. Inicio
2. Registrar reporte
3. Consultar folio
4. Reportes
5. Brigadas
6. Administración
7. Estadísticas
8. Contacto
