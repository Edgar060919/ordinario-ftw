Descripción del proyecto

El proyecto consiste en un sitio web para registrar, consultar y administrar reportes de fallas en el alumbrado público. Utiliza HTML, CSS, JavaScript y archivos XML para cargar información y validar usuarios.

Mockup general

Prompts utilizados

Prompt 1

Diseñar un sitio web de reportes de alumbrado público con mínimo siete páginas.

Prompt 2

Agregar JavaScript para cargar datos desde XML, validar usuarios y filtrar tablas.

Prompt 3

Mejorar el diseño con CSS responsivo y navegación horizontal.
